# CFP-Auto
Carbon Footprint Automation

# 使用方式
編輯 config.yaml 或用 GUI 介面載入 Excel 檔。

按下「開始處理」按鈕。

在 output/ 資料夾裡取回生成的 Word 報告。

功能清單
依據 SimaPro 9.3 版格式自動轉換 Excel raw data

自動計算損壞百分比並內嵌到 Word 範本

Tkinter GUI，支援進度條顯示
